# **App Name**: DataPulse Portfolio

## Core Features:

- Dynamic Project Showcase: Fetch and display data science projects from a backend database or JSON, showcasing key details and metrics.
- Live Demo Integration: Incorporate live demos of data science projects via links or embedded iframes from Streamlit or FastAPI apps.
- Skill-Based Bio: Dynamically render skills, experiences and achievements pulled either from the database or an internal file.
- Interactive Metrics Visualization: Display key project results and metrics through interactive charts and graphs (Chart.js or Recharts).
- Contact Form Submission: Enable users to send messages through a contact form, processed by a Node.js backend to send emails.
- Landing page hero section: Create an engaging hero section with motion effects.
- AI Chatbot Assistant: Implement an AI chatbot powered by a tool that helps answer user queries and provide project information.

## Style Guidelines:

- Primary color: Soft blue (#77B5FE) for a calm, trustworthy feel appropriate to a professional site.
- Background color: Very light desaturated blue (#F0F8FF) to support a light color scheme, and also echoing the trustworthy nature of the content.
- Accent color: Purple (#9370DB) to provide contrast and signal CTAs.
- Body font: 'Inter', a grotesque-style sans-serif for a neutral, machined look, equally good for headlines or body
- Use 'lucide-react' for consistent, clean icons across the site.
- Employ 'shadcn/ui' components for a modern, consistent UI.
- Apply smooth transitions and hover effects using Framer Motion to enhance user experience.
